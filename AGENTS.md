# Agent Instructions

This document provides instructions for AI agents working on this project.

## Getting Started

Before working on the project, make sure you have read the root `README.md` file for an overview of the project and setup instructions.

### Frontend Dependencies

**Important:** Due to limitations in the development environment, running `npm install` in the `apps/frontend` directory might fail. If you encounter issues with frontend dependencies, you may need to run this command manually:

```bash
cd apps/frontend
npm install
```

## Development

When making changes to the code, please adhere to the following guidelines:
